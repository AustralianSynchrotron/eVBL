Capture Tab

The capture tab contains all the functionality regarding the capture and saving of images.

Preview Area

Source:
The selected detector. All connected detectors will be found on starting up the program. If a new detector is connected the software will need to be restarted to be included in the list.

Preview Screen:
Camera feed. Refresh rate can be changed in the settings.

Capture

Single Capture:
Take a single image from the capture and store and display it in the capture area to the right.

Multi Capture:
Take a series of images and take the average. The averaged image will be stored and displayed in the capture area to the right.

File Information

Filename data:
The following data is entered by the user to create the default filename for the image.

  • Initials: Students initials
  • School: Name of the students school
  • Object: The object under investigation or background image
  • Distance: Object distance
  • Wavelength: Wavelength of laser being used
  • Object Number: Number of the diffraction element under investigation

Save Image:
Save the image currently being displayed in the capture area to the right. The default image type is Bitmap. This is recommended as it a raw uncompressed format.

Capture Area

Zoom:
Sets the zoom level of the captured image display.

Capture Area:
The image box on the right side shows the currently buffered image. This is the image which will be saved when the 'Save Image' button is selected.
