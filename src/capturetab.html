Capture Tab

The capture tab contains all the functionality regarding the capture and saving of images.

Preview Area

Source:
The selected detector. All connected cameras will be identified on starting up the program and listed in this drop-down box. If a new camera is connected the software will need to be restarted for it to be included in the list.

Preview Screen:
Live detector feed. The refresh rate can be changed in the settings list.

Capture

Single Capture:
Take a single image from the detector, then store and display it in the display area to the right.

Multi Capture:
Take a series of images and take the average. The averaged image will be stored and displayed in the area to the right.

File Information

Filename data:
The following data is entered by the user to create the default filename for the image.

  • Initials: Students initials
  • School: Name of the students school
  • Object: The object under investigation
  • Distance: Object distance
  • Wavelength: Wavelength of laser being used
  • Object Number: Number of the diffraction element under investigation

Save Image:
Save the image currently being displayed in the capture area to the right. The default image type is tiff. This is recommended as it is saved in a lossless compressed format. For a raw uncompressed format select to save as bitmap.

Capture Area

Zoom:
Sets the zoom level of the captured image display.

Image Display Area:
The image box on the right side shows the currently buffered image. This is the image which will be saved when the 'Save Image' button is selected.
