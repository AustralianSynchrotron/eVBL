Settings

The settings dialog, found in the main menu, is used to configure certain parameters of the software. Under normal use at the Australian Synchrotron Education Centre there would be no need for the student to alter these values.

If these values need to be changed, on saving a file named 'settings.ini' will be created in the program folder. These settings will then be loaded in the future on start-up. It is also possible to edit this file directly in a text editor.

At any time the factory default values can be restored with the 'default' button.



The changeable settings are:
  • mm per pixel: This value is the conversion factor for the line length measurement on the process tab. DEFAULT 0.13760
  • Preview video fps: The frames per second refresh rate of the video capture preview window. Setting this too high will just result in the camera refreshing as fast as it can. Setting to a low value on slower machines may improve performance. DEFAULT 25
  • Intensity line length (pixels): The length in pixels of the intensity profile line on the process tab. DEFAULT 1024
  • Crop box size (pixels): The size of the square box the image will be cropped and saved to in the crop option on the processing tab. DEFAULT 1024
